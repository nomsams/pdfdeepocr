          Troubleshooting & Notes
          
    
            
  • CORS: If you see a generic “Network error,” the server likely blocks browser origins. Use a proxy you control; put its URL in “Proxy URL”.
    • 
            
  • Retries: Enabled by default for HTTP 429/5xx and network errors. Backoff: 1s, 2s, 4s (configurable in code).
    • 
            
  • Watchdog: Detects upload stalls and warns after inactivity (default 15s). You can cancel and retry.
    • 
            
  • Response formats: The app accepts raw Markdown, or JSON containing markdown or text fields.
    • 
            
  • Hashing: SHA-256 is optional; it can be slow for large PDFs on low-end devices.
    • 
            
  • Local settings: Toggle “Remember settings” to persist endpoint, proxy, token, file size limit, and your preferences.
